# Afa Meaning Lookup — Streamlit Community Cloud

## Required repository files

- `app.py`
- `afa_lookup_data.csv`
- `requirements.txt`

Keep all three files in the root of the GitHub repository.

## Deploy

1. Create a GitHub repository.
2. Upload the three files above.
3. Open Streamlit Community Cloud.
4. Create a new app from the repository.
5. Select `app.py` as the entrypoint.
6. In Advanced settings, select Python 3.10.
7. Deploy.

The application does not require API keys or other secrets.
